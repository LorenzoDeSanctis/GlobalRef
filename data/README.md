# GlobalRef Dataset

This dataset accompanies the *GlobalRef* framework and is intended to support
research on temporal synchronization, rigid alignment, and hand–eye calibration
between heterogeneous motion capture systems.

## Contents

- `GlobalRef_DATASET.npz` — motion capture and video-based trajectories used
  for validation and benchmarking.
